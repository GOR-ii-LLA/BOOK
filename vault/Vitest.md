# Vitest
Категория: Claude
Официальная страница: https://vitest.dev
GitHub: https://github.com/vitest-dev/vitest
Подробнее: Vitest — фреймворк для тестирования JavaScript/TypeScript кода, работающий на базе Vite: представь, что у тебя есть помощник, который мгновенно проверяет, правильно ли работает каждая функция в твоём проекте, причём делает это в 5–10 раз быстрее, чем старый стандарт — Jest.
Если твой проект уже использует Vite (а BOOKMARKS именно такой), Vitest подключается без единой строчки дополнительной настройки — он читает тот же vite.config.js и сразу готов к работе, не нужно дублировать конфиги.
Встроенный режим watch мгновенно перезапускает только те тесты, которые затронуты твоими изменениями — как HMR для тестов; плюс есть красивый интерактивный UI (vitest --ui) для наглядного просмотра результатов прямо в браузере.
Для Claude-проектов Vitest особенно полезен при разработке инструментов и скриптов: можно написать тест на парсинг .md файлов из watcher.py-логики, а потом портировать эту логику на JS и убедиться, что всё работает корректно — с моментальной обратной связью.
В 2025 году вышел Vitest 4.0 со стабильным Browser Mode — теперь тесты можно гонять в настоящем Chromium/Firefox/WebKit, а не в симуляции; в 2026 году Vitest стал стандартом де-факто для новых JS-проектов, обогнав Jest по популярности среди разработчиков.
