# claude-orchestration
Категория: Claude
Официальная страница: 
GitHub: https://github.com/mbruhler/claude-orchestration
Подробнее: Это плагин для Claude Code, который позволяет выстраивать цепочки из нескольких ИИ-агентов — представь конвейер на заводе, где каждый рабочий делает свою часть работы, а затем передаёт результат следующему.
Можно описать задачу обычным языком — например, «найди 10 постов с Reddit, проанализируй конкурентов и покажи таблицу» — и плагин сам создаст нужный рабочий процесс, распределит задачи между агентами и покажет прогресс в виде схемы.
Поддерживается параллельное выполнение: несколько агентов работают одновременно, как несколько сотрудников, решающих разные части задачи вместе — это сильно ускоряет сложные проекты.
Удобно использовать вместе с Claude Code для автоматизации больших задач: например, написать тесты → дождаться падения → написать код → проверить → отрефакторить — всё по одной команде.
Проект молодой (198 звёзд, активен в 2026), сообщество только формируется, и часть функций (например, маркетплейс плагинов) пока выглядит экспериментально — стоит тестировать с осторожностью в рабочих проектах.
