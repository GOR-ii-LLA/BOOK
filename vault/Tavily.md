# Tavily

Категория: claude
Официальная страница: https://tavily.com
Подробнее: Tavily — поисковый API, созданный специально для AI-агентов и RAG-систем (Retrieval-Augmented Generation). Возвращает структурированные, очищенные результаты поиска в реальном времени — готовые к подаче в языковую модель без постобработки. Используется разработчиками для подключения LLM-приложений, чат-ботов и AI-агентов к актуальным данным из интернета. Поддерживает поиск, извлечение контента, краулинг сайтов и глубокое исследование; нативно интегрируется с LangChain, LlamaIndex, MCP. В феврале 2026 года Tavily была приобретена компанией Nebius, что вызвало у части разработчиков вопросы о стабильности цен и дальнейшем развитии продукта.
Альтернативы: Exa.ai, Perplexity Sonar API, Brave Search API
