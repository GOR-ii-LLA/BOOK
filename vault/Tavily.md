# Tavily
Категория: Claude
Официальная страница: https://tavily.com
GitHub: https://github.com/tavily-ai
Подробнее: Tavily — это поисковый API, специально созданный для AI-агентов и приложений: он ищет в интернете, извлекает данные со страниц, обходит сайты и проводит исследования, возвращая результат в удобном для ИИ формате.
Главный плюс — скорость и точность: например, если ты строишь чат-бота, который должен отвечать на вопросы с актуальными данными из интернета, Tavily найдёт и отфильтрует нужную информацию за доли секунды (медиана ответа ~180 мс), избавляя от необходимости парсить сайты вручную.
Tavily предлагает сразу несколько инструментов через один API: Search (поиск), Extract (извлечение контента страницы), Crawl (обход сайта), Map (карта структуры сайта) и Research (глубокое исследование темы) — это как целый набор «интернет-помощников» в одной коробке.
Для Claude интеграция особенно удобна: Tavily имеет готовый MCP-сервер, который подключается к Claude Code одной командой и даёт Claude возможность искать в интернете, извлекать данные и обходить сайты прямо во время работы — например, чтобы собрать актуальную информацию о компании перед встречей.
Tavily также интегрирован с LangChain, OpenAI и другими фреймворками, а его MCP-сервер работает с Cursor и Claude Desktop — но стоит учитывать, что бесплатный план ограничен 1000 запросами в месяц, а для серьёзных проектов потребуется платная подписка.
