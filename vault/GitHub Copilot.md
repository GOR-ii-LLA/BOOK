# GitHub Copilot
Категория: Claude
Официальная страница: https://github.com/features/copilot
GitHub: https://github.com/github/copilot-docs
Подробнее: GitHub Copilot — AI-ассистент для написания кода, который встраивается прямо в редактор и помогает писать, исправлять и объяснять код на ходу, как опытный напарник рядом за экраном.
Он умеет дописывать код по контексту (прямо как автодополнение, но умное), отвечать на вопросы в чате («почему эта функция не работает?»), а также самостоятельно редактировать несколько файлов сразу в режиме Agent Mode.
С 2026 года появился Coding Agent: можно назначить GitHub-задачу прямо на Copilot — он сам напишет код, прогонит тесты и откроет Pull Request, пока ты занимаешься другим.
В Claude это полезно как дополнение: Copilot работает внутри VS Code и JetBrains с поддержкой Claude Sonnet как одной из моделей — то есть можно выбирать Claude прямо в Copilot для более вдумчивых задач.
Главный минус — это не бесплатный инструмент для серьёзной работы: бесплатный план ограничен 2000 подсказок в месяц, а Agent Mode быстро съедает лимиты «premium requests».
