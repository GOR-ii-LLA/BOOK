# Serena
Категория: Claude
Официальная страница: https://oraios.github.io/serena
GitHub: https://github.com/oraios/serena
Подробнее: Serena — это MCP-сервер, который даёт Claude «зрение программиста»: вместо того чтобы читать файлы целиком, Claude видит код на уровне символов — находит функции, классы, зависимости между ними, как это делает умная IDE.
Представь, что ты просишь Claude разобраться в большом проекте на тысячи строк — без Serena он «листает» весь код подряд, тратя время и токены; с Serena он сразу переходит к нужной функции и знает, где она используется.
Особенно полезен в сложных проектах: Serena поддерживает более 30 языков программирования — Python, JavaScript, Rust, Go, Java и другие — через стандартный протокол языковых серверов (LSP).
С Claude Code Serena работает как усилитель: Claude сам решает задачи от анализа до коммита, а Serena обеспечивает точную навигацию по коду — меньше ошибок, меньше лишних запросов.
Есть JetBrains-плагин (IntelliJ, PyCharm, WebStorm и др.) как альтернатива LSP — он ещё мощнее, но Serena пока не выпустила стабильный релиз (последний v0.1.4), а значит API может меняться.
