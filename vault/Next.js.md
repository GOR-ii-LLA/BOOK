# Next.js
Категория: Claude
Официальная страница: https://nextjs.org
GitHub: https://github.com/vercel/next.js
Подробнее: Next.js — это популярный фреймворк для создания сайтов и веб-приложений на React, который берёт на себя сложные вещи: настройку сервера, маршрутизацию страниц, загрузку данных и сборку — ты пишешь логику, остальное делает фреймворк.
Главный плюс — скорость и простота старта: одна команда `npx create-next-app` создаёт готовый проект с правильной структурой, конфигурацией TypeScript, Tailwind и AGENTS.md файлом — Claude Code сразу знает, как работать с этим проектом.
Next.js умеет рендерить страницы как на сервере, так и заранее (статически), что делает сайты быстрыми и понятными поисковикам — например, интернет-магазин с тысячью товаров загружается мгновенно, потому что страницы уже готовы.
Начиная с версии 16, Next.js встроил поддержку AI-агентов: Claude Code автоматически читает AGENTS.md и документацию прямо из папки `node_modules/next/dist/docs/` вместо устаревших данных обучения — это резко снижает ошибки при генерации кода; есть также MCP-интеграция, через которую агент видит реальные ошибки браузера и состояние приложения.
Дополнительный сценарий: Claude Code + Next.js — связка для полноценного full-stack проекта без знания бэкенда: API routes, база данных через Prisma, деплой на Vercel одной командой — всё это можно собрать через Claude Code, не написав вручную почти ни строчки.
