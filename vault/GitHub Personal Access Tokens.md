# GitHub Personal Access Tokens
Категория: Claude
Официальная страница: https://github.com/settings/personal-access-tokens
GitHub: 
Подробнее: GitHub Personal Access Tokens (PAT) — это специальные ключи доступа к твоему GitHub-аккаунту, которые используются вместо пароля, когда программы или скрипты обращаются к GitHub от твоего имени.
Например, watcher.py в проекте BOOKMARKS делает git push автоматически — и именно токен позволяет ему войти в GitHub без того, чтобы ты каждый раз вводил пароль вручную.
Существует два вида токенов: классический (даёт широкий доступ ко всем репозиториям) и fine-grained — новый и безопасный, где ты сам выбираешь, к каким именно репозиториям и каким действиям есть доступ, с обязательным сроком действия.
Для интеграции с Claude-проектами (например, чтобы watcher.py мог пушить изменения в BOOKMARKS) достаточно создать fine-grained токен с доступом только к нужному репозиторию и правом «Contents: read/write» — это безопаснее, чем давать полный доступ.
Важный недостаток: токен показывается только один раз при создании — если не скопировал сразу, придётся генерировать заново; также fine-grained токены не работают, если нужен доступ к нескольким организациям одновременно.
