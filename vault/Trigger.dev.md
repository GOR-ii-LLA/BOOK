# Trigger.dev
Категория: Claude
Официальная страница: https://trigger.dev
GitHub: https://github.com/triggerdotdev/trigger.dev
Подробнее: Trigger.dev — это платформа для запуска долгих фоновых задач в TypeScript: представь, что у тебя есть помощник, который берёт на себя всё «медленное» — обработку файлов, отправку писем, запросы к AI — и делает это в фоне, пока твой сайт работает без тормозов.
Главный плюс — никаких таймаутов: задача может работать час, сутки, неделю — столько, сколько нужно, а если что-то пошло не так, она автоматически повторится с того места, где остановилась (как пауза в видео, а не перемотка с начала).
У платформы есть официальный MCP-сервер, который позволяет Claude напрямую запускать и управлять задачами Trigger.dev — например, сказать Claude «запусти обработку этого документа» и он сделает это через MCP, не выходя из чата.
С Claude можно выстроить такой сценарий: Claude принимает запрос пользователя, через MCP запускает длинную цепочку задач в Trigger.dev (например, скачать данные → обработать AI → сохранить результат → отправить письмо), а результат в реальном времени возвращается обратно в интерфейс.
Из минусов: платформа заточена под TypeScript, Python-задачи поддерживаются, но как вторичный гражданин; для простых сценариев (один короткий запрос к AI) это избыточно — Trigger.dev нужен там, где есть реальные цепочки и длинные операции.
