# Strapi
Категория: Claude
Официальная страница: https://strapi.io
GitHub: https://github.com/strapi/strapi
Подробнее: Strapi — это открытая headless CMS на Node.js и TypeScript: она хранит твой контент и отдаёт его по API любому сайту, приложению или устройству, не диктуя, как этот контент должен выглядеть.
Представь, что у тебя есть один склад с текстами, картинками и товарами — а забирать их могут одновременно и сайт, и мобильное приложение, и IoT-устройство, каждый в своём формате: это и есть Strapi.
Strapi поддерживает REST и GraphQL, имеет визуальный конструктор типов контента, гибкую систему ролей и прав, а также растущий маркетплейс плагинов — от email и загрузки файлов до GraphQL и облачных интеграций.
Claude можно подключить к твоему Strapi-проекту через MCP: существует несколько серверов (misterboe/strapi-mcp-server, l33tdawg/strapi-mcp, официальный плагин @sensinum/strapi-plugin-mcp), которые позволяют Claude читать типы контента, создавать и редактировать записи и загружать медиафайлы через естественный язык.
Главный нюанс: MCP-плагины для Strapi предназначены только для разработки и нельзя включать их в продакшн-среде — они открывают внутренние API без должной защиты.
