# Firecrawl
Категория: Claude
Официальная страница: https://firecrawl.dev
GitHub: https://github.com/firecrawl/firecrawl
Подробнее: Firecrawl — это сервис, который превращает любой сайт в чистый текст в формате Markdown, готовый для работы с языковыми моделями: представь, что ты хочешь «скормить» Клоду содержимое какого-то сайта — Firecrawl делает это автоматически, убирая всю лишнюю разметку, рекламу и навигацию.
Умеет парсить динамические сайты на JavaScript, обходить защиту от ботов и одновременно обрабатывать целые разделы сайтов — например, всю документацию по продукту за один запрос.
Позволяет извлекать структурированные данные с помощью простой инструкции на естественном языке — вместо написания кода с CSS-селекторами достаточно написать «вытащи названия и цены товаров».
Имеет официальный MCP-сервер: одна команда `npx -y firecrawl-mcp` — и Клод получает прямой доступ к веб-скрапингу прямо внутри Claude Code или Claude Desktop, без написания кода.
Интегрируется с LangChain и LlamaIndex для создания RAG-систем, а также позволяет автономному агенту (/agent endpoint) самостоятельно исследовать несколько источников в интернете по заданному запросу — однако бесплатный лимит быстро расходуется при активном использовании.
