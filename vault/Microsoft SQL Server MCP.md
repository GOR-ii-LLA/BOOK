# Microsoft SQL Server MCP
Категория: Claude
Официальная страница: https://pypi.org/project/microsoft_sql_server_mcp
GitHub: https://github.com/RichardHan/mssql_mcp_server
Подробнее: MCP-сервер, который соединяет Claude с базой данных Microsoft SQL Server — представь, что ты можешь просто написать Клоду «покажи мне все заказы за март» и он сам напишет SQL-запрос и вернёт ответ, без единой строчки кода с твоей стороны.
Установка занимает буквально одну команду: добавляешь блок с настройками в конфиг Claude Desktop, указываешь адрес сервера, имя базы и пароль — и Клод уже видит твои таблицы.
Поддерживает три способа подключения: обычный SQL-логин, Windows-аутентификацию (удобно в корпоративной сети) и Azure SQL Database — то есть работает и с облачной базой Microsoft.
Главный сценарий для Клода: попросить его изучить структуру базы, потом задавать вопросы на русском — Клод переводит их в SELECT-запросы и возвращает данные; можно также писать INSERT/UPDATE с подтверждением.
Важное ограничение: нельзя давать Клоду права администратора — создай отдельного пользователя с минимальными правами (только чтение, если не нужна запись), иначе при неточном запросе можно случайно изменить или удалить данные.
