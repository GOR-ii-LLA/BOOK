# Playwright
Категория: Claude
Официальная страница: https://playwright.dev
GitHub: https://github.com/microsoft/playwright-mcp
Подробнее: Playwright — инструмент от Microsoft для автоматизации браузера: он умеет открывать сайты, нажимать кнопки, заполнять формы и проверять результаты — как будто за компьютером сидит человек, только в тысячу раз быстрее.
Вместо того чтобы делать скриншот и «смотреть» на него как человек, Playwright читает структуру страницы — что за кнопки есть, какие поля для ввода — и взаимодействует с ними напрямую, без глаз и без лишних затрат на обработку картинок.
Playwright поддерживает три браузерных движка (Chrome, Firefox, Safari) и работает на Windows, Mac и Linux — это значит, что один и тот же скрипт проверит сайт сразу в трёх браузерах без дополнительных усилий.
Claude Code встроил поддержку Playwright CLI как основного инструмента для работы с браузером: когда попросишь Claude «зайди на сайт и проверь», он запустит Playwright и вернёт результат — без скриншотов, экономно по токенам.
Playwright MCP (сервер по протоколу MCP) можно подключить к Claude Desktop одной командой `claude mcp add playwright npx @playwright/mcp@latest` — и тогда Claude сможет буквально управлять браузером прямо во время разговора с тобой.
