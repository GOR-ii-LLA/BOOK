# Git
Категория: Claude
Официальная страница: https://git-scm.com
GitHub: https://github.com/git/git
Подробнее: Git — это система контроля версий: она помнит всю историю изменений твоих файлов, как машина времени — можно вернуться к любой прошлой версии проекта.
Если ты работаешь с Claude Code или пишешь код вместе с Claude, Git сохранит каждый шаг работы: если AI что-то сломает — одна команда откатит всё назад, как будто ничего не было.
96% профессиональных разработчиков используют Git — это стандарт индустрии, без которого не обходится ни один серьёзный проект, даже если над ним работает один человек.
Claude отлично понимает Git: можно описать ситуацию словами («ветка разошлась, как объединить?»), и он объяснит нужную команду или сам выполнит её через Claude Code.
Git работает полностью офлайн — история хранится прямо в папке проекта, GitHub и GitLab нужны только для синхронизации с другими людьми или облачного бэкапа.
