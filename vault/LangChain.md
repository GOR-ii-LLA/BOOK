# LangChain
Категория: Claude
Официальная страница: https://langchain.com
GitHub: https://github.com/langchain-ai/langchain
Подробнее: LangChain — это бесплатный фреймворк для разработчиков, который позволяет создавать приложения и агентов на основе языковых моделей, подключая их к инструментам, базам данных и внешним сервисам.
Представь, что ты хочешь, чтобы Claude мог сам заходить в интернет, читать твои документы и выполнять задачи — LangChain даёт готовые «кубики», из которых это собирается буквально за несколько строк кода.
Фреймворк поддерживает все популярные модели — Claude, GPT, Gemini — и позволяет в любой момент заменить одну на другую, не переписывая остальной код.
Вместе с Claude можно за 10 строк кода создать агента, который умеет использовать инструменты: искать в интернете, читать файлы, выполнять вычисления — и всё это через стандартный интерфейс LangChain.
Важный момент: для сложных многошаговых агентов сами авторы рекомендуют смотреть в сторону LangGraph (отдельный фреймворк из той же экосистемы) — LangChain больше подходит для быстрого старта, а не для сложных рабочих процессов.
