# Cherry Studio
Категория: Claude
Официальная страница: https://cherry-ai.com
GitHub: https://github.com/CherryHQ/cherry-studio
Подробнее: Cherry Studio — это бесплатная десктопная программа, которая объединяет десятки AI-моделей в одном окне: вместо того чтобы держать открытыми claude.ai, ChatGPT и ещё пять вкладок, ты работаешь в одном приложении.
Поддерживает 300+ провайдеров и моделей — Claude, GPT-4o, Gemini, DeepSeek, а также локальные модели через Ollama: можно в одном чате спросить одновременно нескольких AI и сравнить ответы, чтобы выбрать лучший.
Встроена база знаний (RAG): закидываешь туда свои PDF, заметки, документы — и любая модель начинает отвечать с учётом твоих материалов, будто прочитала всё это заранее.
С Claude это особенно удобно: добавляешь API-ключ, подключаешь MCP-серверы прямо из встроенного MCP Marketplace — и Claude получает доступ к файлам, браузеру, поиску без дополнительной настройки.
Из минусов: программа родом из Китая (Shanghai Qianhui Technology), исходный код открыт (AGPL-3.0), но если важна полная прозрачность — стоит это учитывать; кроме того, без API-ключей провайдеров часть функций недоступна.
