# shadcn/ui
Категория: Claude
Официальная страница: https://ui.shadcn.com
GitHub: https://github.com/shadcn-ui/ui
Подробнее: shadcn/ui — это набор готовых блоков интерфейса для React: кнопки, формы, модальные окна, таблицы и ещё 50+ компонентов, которые сразу выглядят красиво и работают для всех пользователей, включая людей с ограничениями.
Главная особенность: ты не устанавливаешь библиотеку как пакет, а копируешь код компонента прямо к себе в проект — командой npx shadcn-ui@latest add button — и дальше делаешь с ним что хочешь, без борьбы с чужими ограничениями.
Это как получить рецепт блюда вместо готовой еды: можно добавить соли, убрать острое, изменить под себя — код полностью твой, никакой зависимости от обновлений чужой библиотеки.
shadcn/ui имеет официальный MCP-сервер, который подключается к Claude Code: достаточно добавить несколько строк конфига, и Клод начинает видеть актуальный код компонентов напрямую из реестра — перестаёт угадывать пропсы и предлагает точный, рабочий код.
Без MCP-сервера Клод работает по памяти из обучения и может предложить устаревшие паттерны или несуществующие пропсы — подключение MCP решает эту проблему и делает совместную работу с Клодом по UI значительно надёжнее.
