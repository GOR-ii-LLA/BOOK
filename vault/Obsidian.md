# Obsidian
Категория: Claude
Официальная страница: https://obsidian.md
GitHub: 
Подробнее: Obsidian — это бесплатная программа для заметок, которая хранит все файлы прямо у тебя на компьютере в виде обычных текстовых .md файлов: никаких облаков, никаких аккаунтов, никто кроме тебя не видит твои записи.
Через Claude Desktop можно подключить Obsidian по протоколу MCP (это как специальный мостик) — и тогда Клод получает доступ к твоим заметкам: может искать по ним, создавать новые, находить связи между идеями, которые ты сам не заметил.
Главная «фишка» — это связи между заметками: ты пишешь [[название другой заметки]] и Obsidian строит визуальную карту, где видно как идеи связаны между собой, — получается что-то вроде твоей личной Википедии.
Если установлен Claude Code, то прямо в боковой панели Obsidian появляется чат с Клодом, который видит твои файлы и может их редактировать, создавать новые заметки, структурировать знания — всё без переключения между программами.
Для связки с Клодом потребуется немного технических шагов: установить плагин-посредник (например, obsidian-mcp-tools или Claudian), и один раз прописать настройки — после этого интеграция работает автоматически.
