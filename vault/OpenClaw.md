# OpenClaw
Категория: Claude
Официальная страница: https://openclaw.ai
GitHub: https://github.com/openclaw-ai/openclaw
Подробнее: OpenClaw — это открытая платформа, которая превращает Claude Code в твоего персонального ИИ-помощника, работающего круглосуточно: он живёт на твоём компьютере или сервере и выполняет задачи самостоятельно, пока ты занимаешься своими делами.
Представь, что у тебя есть умный помощник, которому можно написать в WhatsApp или Telegram — «разбери входящую почту», «проверь билет на рейс», «напомни мне в пятницу» — и он просто сделает это, не требуя твоего присутствия за компьютером.
OpenClaw расширяется «навыками» (skills) — готовыми модулями из магазина ClawHub, которые подключают его к Gmail, Google Calendar, Slack, GitHub, Obsidian и десяткам других сервисов; сообщество постоянно добавляет новые.
Для интеграции с Claude: OpenClaw запускает Claude Code как движок — ты пишешь ему в Discord или Telegram, он выполняет задачи (пишет код, открывает PR, исправляет баги) и присылает результат обратно в чат, даже когда ты далеко от компьютера.
Важный момент на апрель 2026: Anthropic закрыла возможность использовать подписки Claude Pro/Max для OpenClaw — теперь нужна либо оплата по API, либо «Extra Usage»; кроме того, Snyk обнаружил уязвимости в части навыков ClawHub, поэтому рекомендуется запускать OpenClaw в изолированной среде (Docker).
