# FastAPI
Категория: Claude
Официальная страница: https://fastapi.tiangolo.com
GitHub: https://github.com/fastapi/fastapi
Подробнее: FastAPI — это Python-фреймворк для создания веб-API: представь, что тебе нужно написать программу, к которой другие программы будут обращаться с вопросами и получать ответы — FastAPI делает это быстро и надёжно.
Он работает в 3–5 раз быстрее привычного Flask и сопоставим по скорости с Node.js и Go — благодаря этому сайты и сервисы на FastAPI легко выдерживают тысячи одновременных запросов без тормозов.
FastAPI автоматически создаёт интерактивную документацию: открываешь браузер по адресу `/docs` — и видишь живой интерфейс, где можно сразу тестировать все методы API без написания дополнительного кода.
Claude Code имеет специальный скил `fastapi-templates` и умеет генерировать production-ready FastAPI проект одной командой — с async-паттернами, dependency injection и Docker-конфигурацией; а библиотека FastAPI-MCP позволяет автоматически превращать любой FastAPI-эндпоинт в MCP-инструмент для Claude.
Единственный существенный минус — FastAPI не включает ORM, авторизацию и другие «батарейки» из коробки, поэтому для полноценного приложения нужно самостоятельно добавлять SQLAlchemy, аутентификацию и прочие компоненты.
