# Cypress
Категория: Claude
Официальная страница: https://cypress.io
GitHub: https://github.com/cypress-io/cypress
Подробнее: Cypress — это инструмент для автоматического тестирования сайтов: он открывает браузер и сам нажимает на кнопки, заполняет формы, проверяет, что всё работает — так, как это делал бы живой человек.
Это избавляет от ручной проверки после каждого изменения в коде: один раз описываешь сценарий («перейди на страницу, введи логин, проверь, что появилась кнопка»), и Cypress воспроизводит его сколько угодно раз автоматически.
Cypress работает прямо в браузере и не требует сложной настройки — установка через `npm install cypress`, и первый тест можно написать за 10 минут; у него понятный синтаксис на JavaScript, встроенный интерфейс с визуальным воспроизведением каждого шага и удобная отладка с «машиной времени» (можно посмотреть, как выглядел экран на каждом шаге теста).
С Claude Cypress интегрируется через Cloud MCP: Claude подключается к Cypress Cloud и получает прямой доступ к результатам тестов — видит, какие тесты упали, читает ошибки и трассировки стека прямо в чате, без копирования из CI-интерфейса.
Второй сценарий — генерация тестов: Claude Code умеет писать тесты на Cypress по описанию задачи или по коду приложения, а также чинить упавшие тесты, получив от Cloud MCP контекст об ошибке.
Бесплатная версия: Cypress App полностью бесплатен и open-source (MIT). Cypress Cloud — отдельная платформа с платными планами, но есть бесплатный Starter с базовыми функциями; Cloud MCP доступен на всех планах, включая Starter.
