# OpenSpec
Категория: Claude
Официальная страница: https://openspec.dev
GitHub: https://github.com/Fission-AI/OpenSpec
Подробнее: OpenSpec — это бесплатный инструмент, который заставляет Claude (и других AI-помощников) сначала согласовать план с тобой, и только потом писать код — чтобы результат был предсказуемым, а не случайным.
Представь, что ты просишь строителя переделать кухню: без проекта он может снести не ту стену. OpenSpec — это и есть проект: ты описываешь задачу, AI создаёт файлы с планом (что делаем, зачем, как), ты их просматриваешь и подтверждаешь — и только потом начинается реализация.
Инструмент особенно полезен для больших задач в несколько сессий: план хранится прямо в папке проекта в виде Markdown-файлов, и когда откроешь новый чат — Claude сразу видит, на чём остановились, и продолжает без потери контекста.
С Claude Code работает через slash-команды: /opsx:propose "добавить тёмную тему" — и Claude создаёт папку с proposal.md, specs/, design.md и tasks.md; потом /opsx:apply — начинает реализацию; /opsx:archive — закрывает задачу.
Поддерживает 20+ AI-инструментов (Cursor, GitHub Copilot, Windsurf и др.), никакого vendor lock-in, но есть нюанс: это не MCP-сервер в классическом смысле — MCP Market разместил его как «MCP-совместимый», однако по сути это CLI-утилита и набор скиллов для Claude Code, а не сервер с инструментами.
