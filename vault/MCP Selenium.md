# MCP Selenium
Категория: Claude
Официальная страница: https://npmjs.com/package/@angiejones/mcp-selenium
GitHub: https://github.com/angiejones/mcp-selenium
Подробнее: MCP Selenium — это сервер по протоколу MCP (Model Context Protocol), который даёт Claude прямые руки для управления браузером: открыть Chrome, перейти на сайт, нажать кнопку, ввести текст, сделать скриншот — всё это словами, без написания кода.
Представь, что тебе нужно каждый день заходить на сайт, нажимать три кнопки и копировать таблицу в Excel — Claude сделает это сам, если ему подключить этот инструмент.
Особенно полезен тем, кто пишет тесты для сайтов: вместо того чтобы вручную описывать каждый шаг на Python, ты говоришь Клоду «проверь, что кнопка входа работает» — и он сам генерирует и выполняет нужные действия.
Подключается одной строкой в настройках Claude Desktop — npx запускает сервер автоматически, без установки дополнительных программ вручную; поддерживает Chrome, Firefox и Edge.
Из ограничений: Safari работает только на Mac и требует ручной настройки; проект активен, 338 звёзд на GitHub, но это небольшой open-source, а не корпоративный продукт.
