# Claude Cookbooks
Категория: Claude
Официальная страница: 
GitHub: https://github.com/anthropics/claude-cookbooks
Подробнее: Это официальная библиотека от Anthropic — коллекция готовых рецептов и примеров кода, которая показывает, как использовать Claude для реальных задач: от суммаризации текста до создания агентов и работы с памятью.
Каждый рецепт — это Jupyter-ноутбук с живым кодом на Python, который можно скопировать и сразу запустить: например, готовый пример классификации текста или генерации Excel-файла через API буквально за несколько минут.
В репозитории есть раздел Skills — готовые «пакеты экспертизы», которые Claude может загружать и использовать для создания документов Word, PowerPoint, PDF, а также для бизнес-автоматизации — всё с примерами кода.
Если ты хочешь научить Claude чему-то своему — здесь есть шаблон кастомного скила (папка с SKILL.md, скриптами и ресурсами), который легко адаптировать под любую задачу и подключить к своему проекту через API.
Репозиторий активно развивается и принимает вклад сообщества — если у тебя есть идея нового рецепта, её можно предложить через Issues, однако некоторые темы уже хорошо покрыты похожими ресурсами вроде официальной документации.
