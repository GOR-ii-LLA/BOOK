# Archon
Категория: Claude
Официальная страница: 
GitHub: https://github.com/coleam00/Archon
Подробнее: Archon — это MCP-сервер и веб-приложение, которое даёт Claude Code и другим AI-инструментам доступ к твоей личной базе знаний: загружаешь документацию, PDF-файлы, сайты — и Claude начинает работать с ними как с контекстом.
Например, ты работаешь над проектом и хочешь, чтобы Claude знал всю документацию твоего фреймворка — просто вставляешь ссылку в Archon, он обходит сайт и сохраняет всё в базу, а Claude теперь всегда может к ней обратиться.
Внутри есть удобный интерфейс для управления задачами и проектами: можно создавать задачи, а AI сам помогает их расписать и выстроить в логику — это экономит время при старте любого проекта.
Для интеграции с Claude Code достаточно скопировать конфиг подключения из дашборда и вставить в настройки — после этого Archon становится MCP-сервером, через который Claude получает доступ к твоим материалам в любой момент работы.
Серьёзный минус: проект в бета-версии и требует Docker + Supabase + OpenAI API для работы — это несколько шагов настройки, которые могут быть сложноваты для новичков.
