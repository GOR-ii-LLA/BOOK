# n8n-MCP
Категория: Claude
Официальная страница: https://n8n-mcp.com
GitHub: https://github.com/czlonkowski/n8n-mcp
Подробнее: n8n-MCP — это MCP-сервер, который даёт Клоду полную документацию по платформе автоматизации n8n: он знает все 1396 нод (блоков-действий), их свойства и операции, поэтому может строить рабочие автоматизации без ошибок и догадок.
Представь, что хочешь настроить автоматику: «когда приходит письмо — создай задачу в Notion и отправь уведомление в Telegram». Без MCP Клод угадывает названия параметров и часто ошибается; с n8n-MCP он точно знает какие поля нужны — и всё работает с первого раза.
Сервер содержит 2646 готовых примеров конфигураций из реальных шаблонов, что позволяет Клоду предлагать проверенные решения, а не изобретать велосипед.
Перед отправкой схемы в n8n сервер проверяет конфигурацию на ошибки — особенно удобно для сложных AI-агент-воркфлоу, где один неправильный параметр ломает всю цепочку в 3 часа ночи.
Работает с Claude Desktop, Claude Code, Cursor и Windsurf через npx n8n-mcp — установка занимает 5 минут; подключение к реальному инстансу n8n требует API-ключ, без него сервер работает только как справочник по нодам.
