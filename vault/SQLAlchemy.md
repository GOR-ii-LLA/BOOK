# SQLAlchemy
Категория: Claude
Официальная страница: https://sqlalchemy.org
GitHub: https://github.com/sqlalchemy/sqlalchemy
Подробнее: SQLAlchemy — это Python-библиотека для работы с базами данных: она позволяет не писать SQL-запросы вручную, а работать с данными как с обычными объектами Python — например, вместо `INSERT INTO ...` ты просто создаёшь объект и говоришь «сохрани».
Главный плюс — универсальность: она умеет работать с PostgreSQL, MySQL, SQLite, Oracle, Microsoft SQL Server и другими базами через единый интерфейс, а значит можно переключиться на другую базу почти без правок в коде.
Внутри — два уровня: ORM (высокий, удобный — ты работаешь с объектами и классами) и Core (низкий, гибкий — для тех, кому нужен точный контроль над SQL-запросами).
С Claude это работает идеально в паре: ты описываешь словами что нужно сделать с данными, Claude пишет код на SQLAlchemy — создаёт таблицы, фильтрует, сортирует, связывает; это особенно мощно при построении FastAPI-приложений или MCP-серверов, где Claude через SQLAlchemy может читать и писать в твою базу данных.
Реальный сценарий: Claude Code открывает твой проект, видит модели SQLAlchemy, понимает структуру базы и сам пишет запросы под новую задачу — без объяснений что такое таблицы и поля.
