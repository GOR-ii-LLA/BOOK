# Docker
Категория: Claude
Официальная страница: https://docker.com
GitHub: https://github.com/QuantGeekDev/docker-mcp
Подробнее: Docker — это программа, которая упаковывает любое приложение в изолированный «контейнер», и этот контейнер одинаково работает на любом компьютере, будь то твой ноутбук или сервер в облаке.
В связке с Claude это превращается в мощную рабочую среду: Docker MCP Toolkit даёт Клоду доступ к реальным инструментам — GitHub, Jira, файловой системе — и он может не просто советовать, а буквально делать: создавать задачи, пушить код, анализировать репозитории.
Благодаря Docker Model Runner можно запускать языковые модели локально прямо на своём компьютере, и Claude Code будет работать с ними как с обычным API — без отправки данных в интернет, дёшево и под полным контролем.
Docker Sandboxes позволяют запустить Клода в полностью изолированной среде — он может свободно устанавливать пакеты, менять файлы и запускать контейнеры, не рискуя сломать твой компьютер: всё происходит внутри «песочницы».
Серьёзный порог входа: Docker требует понимания контейнеров, сетей и командной строки — без этого базового опыта настройка MCP Toolkit или Sandboxes может занять несколько часов.
