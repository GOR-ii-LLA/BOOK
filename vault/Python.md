Подробнее: Python — язык программирования, на котором пишут скрипты, автоматизируют рутину, создают сайты, работают с данными и обучают нейросети; именно на нём написан watcher.py в этом проекте.
Это самый популярный язык в AI-разработке — Claude Code, Anthropic SDK, LangChain — всё это Python, и понимание основ помогает читать и адаптировать чужие скрипты под себя.
Синтаксис Python специально сделан простым и читаемым: код выглядит почти как обычный текст, поэтому новичок может разобраться в чужом скрипте даже без опыта программирования.
Когда Claude пишет скрипт для автоматизации — разбора файлов, отправки запросов к API, обработки данных — он почти всегда пишет его на Python, и иметь Python установленным на компьютере означает запустить этот скрипт одной командой.
Единственный серьёзный момент: разные версии Python (3.10, 3.12, 3.14) иногда несовместимы с библиотеками, и если скрипт не запускается — причина часто именно в версии; решается установкой нужной через python.org/downloads.
