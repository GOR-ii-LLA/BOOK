# Ruflo
Категория: Claude
Официальная страница: https://mcpmarket.com/server/ruflo
GitHub: https://github.com/ruvnet/ruflo
Подробнее: Ruflo (бывший Claude Flow) — это MCP-сервер и платформа оркестрации, которая превращает Claude Code из одиночного ИИ-помощника в целую команду специализированных агентов, работающих параллельно над одной задачей.
Представь, что вместо одного человека над твоим кодом работают сразу 8 — один анализирует архитектуру, другой пишет тесты, третий проверяет безопасность: именно так работает Ruflo, запуская до 60+ специализированных агентов-«роев» одновременно.
Ruflo самообучается — он запоминает удачные решения из прошлых задач и применяет их снова, а умная маршрутизация автоматически выбирает более дешёвую модель для простых задач, экономя до 75% затрат на API.
Прямо из Claude Code можно одной командой инициализировать рой агентов: claude mcp add ruflo -- npx -y ruflo@latest mcp start — и Claude получит доступ к 313 инструментам для координации агентов, векторной памяти, нейронного обучения и сканирования безопасности.
Платформа активно развивается (v3.5.0, более 5800 коммитов, ~500 000 скачиваний) и работает не только с Claude, но и с GPT, Gemini и локальными LLM — однако это же означает высокую сложность: для новичка настройка потребует времени.
