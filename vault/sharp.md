# sharp
Категория: Claude
Официальная страница: https://sharp.pixelplumbing.com
GitHub: https://github.com/lovell/sharp
Подробнее: Sharp — это библиотека для Node.js, которая умеет быстро обрабатывать изображения: менять размер, обрезать, конвертировать форматы и многое другое, работая через мощную C-библиотеку libvips под капотом.
Главный плюс — скорость: sharp обрабатывает картинки в 4–5 раз быстрее, чем популярные альтернативы вроде ImageMagick, при этом почти не нагружает память — обрабатывает изображение небольшими кусочками, а не целиком.
Поддерживает все популярные форматы: JPEG, PNG, WebP, AVIF, GIF, TIFF и SVG; это значит, что можно, например, принять загруженную пользователем фотографию и автоматически сохранить её в компактный WebP для сайта — всего несколькими строками кода.
С Claude Code sharp удобно использовать для автоматизации рутины: попросить Claude написать скрипт, который пройдётся по папке с фотографиями и конвертирует все в нужный размер и формат — это делается буквально за одну команду.
Серьёзных недостатков нет, библиотека активно развивается (последнее обновление — ноябрь 2025), но требует Node.js 18.17+ и при установке скачивает бинарные файлы libvips, что иногда вызывает трудности в закрытых корпоративных окружениях.
