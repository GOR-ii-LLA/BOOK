# MCP SQLite
Категория: Claude
Официальная страница: https://github.com/modelcontextprotocol/servers/tree/main/src/sqlite
GitHub: https://github.com/modelcontextprotocol/servers
Подробнее: MCP SQLite — это официальный MCP-сервер от Anthropic, который даёт Клоду прямой доступ к локальным базам данных SQLite прямо в разговоре — как если бы ты показал другу таблицы в Excel и попросил с ними поработать.
Клод умеет читать структуру таблиц, выполнять запросы на выборку, добавлять и редактировать данные — например, «покажи все покупки за март» или «найди дубликаты в таблице клиентов».
Встроенная функция business-insights: Клод сам формирует «аналитическую заметку» и обновляет её по ходу разговора — удобно, если нужно разобраться в данных без написания SQL вручную.
Особенно полезно в связке с MCP Filesystem: можно импортировать CSV-файл, загнать в SQLite и сразу попросить Клода проанализировать — всё в одном разговоре.
Сервер помечен как архивный (archived) — Anthropic переместил его в отдельный репозиторий, однако он стабильно работает; для большинства задач лучше использовать его через `uv` или Docker по официальной инструкции.
