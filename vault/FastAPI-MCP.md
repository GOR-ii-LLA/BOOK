# FastAPI-MCP
Категория: Claude
Официальная страница: https://fastapi-mcp.tadata.com
GitHub: https://github.com/tadata-org/fastapi_mcp
Подробнее: FastAPI-MCP — это библиотека, которая автоматически превращает любой твой FastAPI-сервер (то есть готовый сайт или API на Python) в набор инструментов для Claude — буквально двумя строчками кода, без ручной настройки.
Представь, что у тебя есть небольшой сайт с каталогом товаров на FastAPI: FastAPI-MCP сам «видит» все твои адреса (эндпоинты) и делает их доступными для Claude, чтобы он мог, например, искать товары или оформлять заказы прямо из чата.
Проект активно развивается: 11 600+ звёзд на GitHub, поддержка авторизации через стандартные FastAPI-механизмы, работает без лишних HTTP-вызовов — общается с твоим приложением напрямую через ASGI, что быстрее и надёжнее.
Если ты работаешь с Claude Code или Claude Desktop и у тебя уже есть FastAPI-приложение — FastAPI-MCP позволяет за пять минут подключить его как MCP-сервер и дать Claude доступ ко всем его возможностям.
Серьёзных недостатков нет, проект живой и активно поддерживается; единственное ограничение — требует Python 3.10+ и знакомства с FastAPI, то есть нужен хотя бы базовый опыт разработки.
