# Remotion
Категория: Claude
Официальная страница: https://remotion.dev
GitHub: https://github.com/remotion-dev/remotion
Подробнее: Remotion — это инструмент, который позволяет создавать настоящие видео-файлы (.mp4) с помощью кода на React: вместо того чтобы двигать клипы мышкой в монтажном редакторе, ты описываешь каждый кадр как обычный React-компонент.
Это открывает мощную возможность: один шаблон видео может автоматически генерировать тысячи персонализированных роликов — например, компания создаёт один компонент «обзор недвижимости», передаёт в него данные 100 объектов, и получает 100 уникальных видео за 15 минут без ручного монтажа.
Для визуализации данных это незаменимо: подключи API или CSV, передай данные в Remotion-компонент — и получи анимированный дашборд с растущими графиками, цифрами и метриками в виде полноценного HD-видео.
В январе 2026 Remotion запустил интеграцию Agent Skills для Claude Code: устанавливаешь скилл одной командой (`npx skills add remotion-dev/skills`), затем просто описываешь видео в Claude на обычном языке — Claude генерирует готовый React-код, а Remotion рендерит его в MP4.
Серьёзный нюанс: для небольших команд (до 3 человек) Remotion бесплатен, но командам от 4 человек требуется платная лицензия от $100/мес — это важно учитывать при коммерческом использовании.
