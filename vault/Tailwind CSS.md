# Tailwind CSS
Категория: Claude
Официальная страница: https://tailwindcss.com
GitHub: https://github.com/tailwindlabs/tailwindcss
Подробнее: Tailwind CSS — это CSS-фреймворк, где ты стилизуешь элементы прямо в HTML короткими классами вместо написания отдельных CSS-файлов: хочешь синюю кнопку — пишешь `bg-blue-500`, хочешь отступ — `mt-4`.
Claude и другие AI-инструменты пишут Tailwind особенно хорошо, потому что всё стилизование находится в одном месте — в разметке, без необходимости прыгать между файлами и разгадывать конфликты стилей.
Версия v4 (2025) полностью переписана с нуля: конфигурация переехала из JavaScript-файла в сам CSS, а сборка стала в 5 раз быстрее за счёт нового движка на Rust.
Отличный сценарий с Claude: попроси «сделай адаптивную карточку товара» — Claude напишет весь HTML с Tailwind-классами сразу, без отдельных CSS-файлов, и результат можно сразу вставить в проект.
Важный нюанс: создатель Tailwind сам тестировал Claude Code для массового добавления тёмной темы и нашёл его на 50% медленнее ручной работы — AI хорош для новых компонентов, но для сложного рефакторинга больших проектов лучше разбивать задачи на мелкие части.
