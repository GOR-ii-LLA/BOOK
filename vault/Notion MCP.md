# Notion MCP
Категория: Claude
Официальная страница: https://developers.notion.com/guides/mcp/mcp
GitHub: https://github.com/makenotion/notion-mcp-server
Подробнее: Notion MCP — официальный сервер от Notion, который позволяет Claude читать и писать в твоё рабочее пространство Notion на живом языке, как будто у него есть руки внутри твоих заметок и баз данных.
Можно сказать Клоду: «Создай технический документ по задаче из этой страницы» — и он сам откроет нужную страницу, соберёт контекст и напишет документ прямо в Notion.
Поиск тоже работает голосом: «Найди все мои заметки про проект X» — Клод прошерстит весь воркспейс и вернёт результат, без ручного листания.
В Claude Desktop и Claude.ai подключение через OAuth в один клик — нужно просто зайти в настройки Notion и разрешить доступ; в Claude Code одна команда: claude mcp add --transport http notion https://mcp.notion.com/mcp.
Серьёзный нюанс: Клод действует с твоими полными правами доступа, то есть он видит и может изменить всё то, что видишь ты — важно понимать, что ты ему разрешаешь делать.
