# Superpowers
Категория: Claude
Официальная страница: https://mcpmarket.com/server/superpowers
GitHub: https://github.com/obra/superpowers
Подробнее: Superpowers — это плагин для Claude Code, который учит его работать как опытный разработчик: сначала разобраться в задаче, потом планировать, и только затем писать код.
Представь, что вместо того чтобы сразу бросаться писать код, Claude сначала задаёт тебе уточняющие вопросы, составляет чёткую спецификацию и показывает её тебе по кускам — как хороший консультант перед началом работы.
После твоего одобрения плагин составляет детальный план реализации с упором на тест-разработку (TDD): сначала пишутся тесты, которые проверяют, работает ли код, а потом уже сам код — это сильно снижает число ошибок.
Плагин умеет запускать подагентов — отдельные «рабочие» экземпляры Claude, которые параллельно выполняют задачи, проверяют код и соблюдают план, что особенно полезно при разработке сложных многофайловых проектов в Claude Code.
Продукт очень активно развивается (более 130 тыс. установок, 7,8 тыс. звёзд на GitHub, обновление выходило 3 дня назад), но работает только в среде кодирующих агентов — Claude Code, Codex, Copilot CLI, OpenCode — не в обычном чате Claude.
