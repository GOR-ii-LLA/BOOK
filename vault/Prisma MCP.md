# Prisma MCP
Категория: Claude
Официальная страница: https://prisma.io/mcp
GitHub: https://github.com/prisma/mcp
Подробнее: Prisma MCP — это сервер протокола MCP, который позволяет Claude и другим AI-агентам управлять базами данных на естественном языке: создавать таблицы, запускать миграции, делать резервные копии — просто описывая задачу словами.
Например, можно написать «создай новую базу данных в регионе us-east-1 для этого проекта» — и Claude сам выполнит нужные команды через Prisma CLI, не требуя знания SQL или терминала.
Есть два режима: локальный сервер — для работы на своём компьютере с локальной базой, и удалённый — для платформ, где нужно дать пользователям доступ к управлению базами данных.
Подключить к Claude Desktop или Claude Code можно за 2 минуты: добавить одну строчку JSON в конфиг, и Claude увидит все доступные инструменты — проверку миграций, открытие Prisma Studio, выполнение SQL-запросов.
Работает также с Cursor, Windsurf, Warp, VS Code и ChatGPT — то есть подключается один раз, и доступен во всех инструментах сразу.
