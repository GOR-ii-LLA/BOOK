# pgEdge Postgres MCP
Категория: Claude
Официальная страница: https://pgedge.com/blog/introducing-the-pgedge-postgres-mcp-server
GitHub: https://github.com/pgEdge/pgedge-postgres-mcp
Подробнее: Это MCP-сервер, который позволяет Клоду напрямую общаться с базой данных PostgreSQL на обычном языке — ты пишешь вопрос, Клод сам составляет SQL-запрос и возвращает результат.
Работает с любой PostgreSQL 14+ — твоей локальной, Amazon RDS или облачной pgEdge Cloud, не нужно создавать специальную базу под него.
Клод видит полную схему базы: таблицы, связи между ними, индексы, ограничения — это значит, он понимает структуру данных и не гадает вслепую при составлении запросов.
Удобный сценарий: подключить к Claude Code свою рабочую или тестовую базу данных и просить Клода находить медленные запросы, объяснять структуру таблиц, генерировать миграции или просто отвечать на вопросы по данным на человеческом языке.
Осторожно: сервер в режиме GA, но предоставляет Клоду доступ ко всей схеме и данным — подходит только для внутренних инструментов и доверенных окружений, не для публичных приложений.
