# Task Master
Категория: Claude
Официальная страница: https://github.com/eyaltoledano/claude-task-master
GitHub: https://github.com/eyaltoledano/claude-task-master
Подробнее: Task Master — это AI-менеджер задач для разработчиков: ты пишешь описание проекта обычным языком (так называемый PRD — «что хочу сделать»), а Task Master сам разбивает его на пронумерованные задачи с зависимостями, уровнями сложности и подзадачами.
Вместо того чтобы скармливать Claude или Cursor весь проект целиком (что ведёт к путанице и ошибкам), Task Master подаёт агенту ровно одну задачу за раз — разработчики сообщают о снижении ошибок до 90%.
Подключается как MCP-сервер прямо в редактор (Cursor, Windsurf, VS Code, Claude Code) — после настройки агент сам читает, создаёт и отмечает задачи как выполненные, не выходя из твоей среды разработки.
Режим «core» загружает только 7 ключевых инструментов вместо 36 — это даёт экономию токенов около 76%, что важно при работе с платными API.
Есть режим autopilot для TDD: Task Master сам генерирует тест, пишет код, проверяет, делает коммит — и повторяет по кругу без участия человека; из минусов — нужен хотя бы один ключ API стороннего провайдера, бесплатной автономной работы нет.
