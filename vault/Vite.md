# Vite
Категория: Claude
Официальная страница: https://vitejs.dev
GitHub: https://github.com/vitejs/vite
Подробнее: Vite (читается «вит», от французского «быстро») — это инструмент сборки для веб-разработки: он запускает проект на твоём компьютере во время работы над кодом и упаковывает всё в готовый файл для публикации в интернете.
Главная фишка — мгновенный старт: когда ты меняешь строчку кода, браузер обновляет только эту часть страницы за доли секунды, не перезагружая всё — как будто редактируешь живой документ.
Работает с React, Vue, TypeScript и другими популярными технологиями «из коробки», без сложных настроек — достаточно одной команды `npm create vite@latest`, и рабочий проект готов за минуту.
Claude Code отлично понимает проекты на Vite: он читает `vite.config.js`, видит структуру файлов и может самостоятельно добавлять плагины, настраивать пути и разбираться в ошибках сборки — это стандартная связка для AI-разработки в 2025 году.
Сам BOOKMARKS работает именно на Vite — это и есть то, что превращает твой React-код в сайт на GitHub Pages; без него ничего бы не заработало.
