# Astro
Категория: Claude
Официальная страница: https://astro.build
GitHub: https://github.com/withastro/astro
Подробнее: Astro — это бесплатный фреймворк для создания сайтов, который по умолчанию не грузит в браузер почти никакого JavaScript, что делает страницы быстрыми как молния — особенно это важно для блогов, портфолио, лендингов и документации.
Представь, что ты строишь сайт из конструктора LEGO: Astro позволяет использовать компоненты из React, Vue, Svelte и других популярных инструментов одновременно на одной странице — выбирай любой привычный набор.
Сайты на Astro стабильно получают 100 баллов в Google Lighthouse (это оценка скорости и качества сайта), потому что фреймворк убирает всё лишнее на этапе сборки — в итоге страница весит на 90% меньше, чем аналог на React.
Claude Code идеально работает с Astro: файлы контента хранятся в обычном Markdown, который ИИ читает и редактирует напрямую — не нужен никакой CMS-интерфейс, можно просто говорить с Claude, и он создаёт или обновляет страницы.
У Astro есть официальный MCP-сервер (mcp.docs.astro.build/mcp), который подключается к Claude и даёт ему доступ к актуальной документации прямо во время разработки — это особенно полезно, так как ИИ без MCP может использовать устаревшие API.
