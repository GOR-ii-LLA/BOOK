# RTK (Rust Token Killer)
Категория: Claude
Официальная страница: https://rtk-ai.app
GitHub: https://github.com/rtk-ai/rtk
Подробнее: RTK — это маленькая программа-прокси на Rust, которая перехватывает вывод терминальных команд и сжимает его в 6–10 раз до того, как он попадёт в контекстное окно Claude или другого AI-агента.
Представь: ты просишь Claude запустить тесты, а `cargo test` вываливает 4823 токена успешных результатов — RTK превращает это в одну строчку «262 passed» за 11 токенов, экономя 99% контекста.
То же самое работает с `git status`, `git diff`, `git log`, `ls`, `grep`, `find` — более 30 команд: вместо многостраничного мусора Claude видит только суть.
Один раз запускаешь `rtk init --global`, и всё — RTK автоматически перехватывает нужные команды через хук Claude Code (PreToolUse), работая полностью прозрачно: Claude Code не знает о подмене, он просто получает сжатые данные.
По данным разработчика, за типичную 2-часовую сессию без RTK CLI-команды генерируют ~210 000 токенов шума — с RTK это падает до ~23 000, сессии работают в 3 раза дольше, а на платных API-планах счёт снижается примерно на 70%.
