# mcp-server-mysql
Категория: Claude
Официальная страница: https://smithery.ai/server/@benborla29/mcp-server-mysql
GitHub: https://github.com/benborla/mcp-server-mysql
Подробнее: mcp-server-mysql — это MCP-сервер, который даёт Claude прямой доступ к твоей базе данных MySQL: ты пишешь на обычном языке «покажи всех пользователей, у которых нет заказов за последние 30 дней», а Claude сам составляет SQL-запрос и возвращает результат.
Это как нанять переводчика между тобой и базой данных — больше не нужно знать SQL, чтобы получить нужные данные; достаточно описать задачу словами прямо в диалоге с Claude.
По умолчанию сервер работает только на чтение (SELECT), так что случайно что-то сломать или удалить невозможно; запись (INSERT, UPDATE, DELETE) включается отдельно и только если это действительно нужно.
Хорошо работает в связке с Claude Code: запускаешь сервер локально командой `claude mcp add`, указываешь данные подключения к своей MySQL — и в терминале появляется новый инструмент для работы с базой прямо в процессе разработки.
Среди минусов: настройка требует терминала и базовых знаний командной строки; версия 2.0.5 имеет известный баг с молчаливым завершением, рекомендуется 2.0.2 или ждать исправления.
