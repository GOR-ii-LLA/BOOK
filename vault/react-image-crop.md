# react-image-crop
Категория: Claude
Официальная страница: https://npmjs.com/package/react-image-crop
GitHub: https://github.com/dominictobias/react-image-crop
Подробнее: Это маленькая библиотека для React, которая добавляет в приложение интерактивный инструмент обрезки изображений — пользователь мышью выделяет нужную область, и ты получаешь её координаты или готовый canvas.
Весит меньше 5 КБ в сжатом виде и не тянет никаких сторонних зависимостей — просто добавляешь один компонент <ReactCrop> и оборачиваешь им тег <img>.
Поддерживает фиксированное соотношение сторон (например, 1:1 для аватарок или 16:9 для обложек), минимальный и максимальный размер выделения, круглую область кадрирования, управление с клавиатуры и тачскрин.
Сценарий интеграции с Claude: Claude генерирует компонент загрузки и обрезки аватара — пользователь загружает фото, обрезает нужную область, библиотека возвращает координаты, и Claude помогает дописать функцию экспорта в base64 для отправки на сервер.
Библиотека намеренно сделана минималистичной и не умеет вращать, фильтровать или аннотировать — для этого авторы рекомендуют платный Pintura; последний релиз (v11) — апрель 2025, проект активен.
