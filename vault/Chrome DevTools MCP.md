# Chrome DevTools MCP
Категория: Claude
Официальная страница: https://github.com/ChromeDevTools/chrome-devtools-mcp
GitHub: https://github.com/ChromeDevTools/chrome-devtools-mcp
Подробнее: Chrome DevTools MCP — это официальный MCP-сервер от Google, который даёт Claude и другим ИИ-ассистентам прямой доступ к Chrome: видеть страницу, нажимать кнопки, читать ошибки — всё то, что обычно делает только человек за компьютером.
Представь, что ты просишь Claude починить баг на сайте — раньше он «вслепую» менял код и просил тебя проверить руками, теперь он сам открывает браузер, воспроизводит проблему, видит ошибку в консоли и сразу её исправляет.
Claude может перехватывать сетевые запросы и видеть, почему API возвращает ошибку, делать скриншоты страницы, читать консоль прямо во время работы — это особенно ценно при отладке сложных сценариев с авторизацией или стриминговыми ответами.
Мощный сценарий для разработки с Claude Code: подключаешь сервер один раз, затем просто говоришь «открой localhost:3000 и проверь форму оплаты» — Claude сам проходит весь сценарий, замечает что сломалось и пишет фикс.
Важное ограничение: сервер официально поддерживает только Google Chrome и Chrome for Testing; по умолчанию запускает изолированный профиль без твоих логинов — для работы с сайтами, требующими авторизации, нужна отдельная настройка через флаг --user-data-dir.
