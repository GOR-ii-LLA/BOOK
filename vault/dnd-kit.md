# dnd-kit
Категория: Claude
Официальная страница: https://dndkit.com
GitHub: https://github.com/clauderic/dnd-kit
Подробнее: dnd-kit — это современный инструментарий для создания интерфейсов перетаскивания (drag and drop) в React и других фреймворках: Vue, Svelte, Solid, а также в чистом JavaScript.
Представь, что ты строишь доску задач в стиле Trello, где карточки можно перетаскивать между колонками — dnd-kit даёт тебе все нужные крючки (хуки) и компоненты, чтобы это заработало красиво и плавно без написания сложной логики с нуля.
Библиотека весит всего около 10 КБ, работает с мышью, тачскрином и клавиатурой, поддерживает экранные читалки (accessibility), а анимации настолько плавные, что пользователь ощущает физику перемещения.
С Claude можно за несколько минут сгенерировать сортируемый список, канбан-доску или drag-and-drop конструктор страниц — Claude хорошо знает API dnd-kit и умеет писать компоненты с DndContext, useDraggable, useDroppable и @dnd-kit/sortable.
Из минусов: не работает с HTML5 Drag and Drop API, а значит нельзя перетаскивать файлы с рабочего стола в браузер — для таких задач нужны другие инструменты.
